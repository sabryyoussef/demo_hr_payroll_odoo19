# User Walkthrough

## 1. Open Dashboard

Menu:

```text
Payroll Demo Enterprise > Operations Dashboard
```

Say:

"This is the payroll control center. HR, finance and managers can see the operational workload before payroll is closed."

## 2. Review Attendance Alerts

Click:

```text
Attendance Alerts
```

Say:

"Before payroll, attendance exceptions must be reviewed. Missing check-outs, overtime and late check-ins are visible without manual spreadsheet tracking."

## 3. Run Mass Attendance Adjustment

Menu:

```text
Payroll Demo Enterprise > Operations > Mass Operations
```

Create/run:

```text
Batch Attendance Adjustment
```

Say:

"This is the mass operations message: one operation reviews many employee records."

## 4. Generate Payroll

Click dashboard button:

```text
Generate Payroll
```

Say:

"Odoo creates the payroll batch and payslips for the selected population."

## 5. Create Bank Transfer

Click:

```text
Create Bank Transfer
```

Say:

"Payroll is converted into a finance-ready bank transfer batch."

## 6. Reconcile Payroll

Open:

```text
Reports > Payroll Reconciliation
```

Say:

"Finance can see matched transfers and exceptions. This closes the loop between HR, payroll, bank payment and accounting."
